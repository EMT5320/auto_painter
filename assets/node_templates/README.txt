# assets/node_templates/
# 存放各节点类型的模板图片，供模板匹配识别使用
#
# 目录结构：
#   node_templates/
#     monster/     # 普通敌人节点截图
#     elite/       # 精英节点截图
#     rest/        # 休息/营火节点截图
#     merchant/    # 商人节点截图
#     unknown/     # 未知/问号节点截图
#     treasure/    # 宝箱节点截图
#     boss/        # Boss 节点截图
#
# 获取方式：从游戏地图截图中裁剪对应图标，保存为 PNG 格式（透明背景最佳）
# 建议尺寸：与游戏实际节点图标等大（约 40-80 px）
