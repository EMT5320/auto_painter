# 🎨 Auto Painter — 自动绘画程序

在杀戮尖塔2地图界面，按住右键可以自由绘画。  
本工具可以自动模拟这个过程，将**文字或图片**绘制到画面上。

---

## 安装依赖

```bash
pip install -r requirements.txt
```

## 启动

```bash
python main.py
```

---

## 使用流程

1. 运行程序，选择模式（图片 / 文字）
2. 按提示输入内容和参数
3. 设置倒计时秒数（默认5秒），期间切换到游戏窗口并停留在地图界面
4. 程序自动开始绘制，以屏幕中心为基准，占屏幕约 60%

> ⚡ **紧急停止**：把鼠标移到屏幕**左上角**立即中断

---

## 模式说明

### 🖼 图片模式
- 支持 jpg / png / bmp 格式
- 使用 Canny 边缘检测提取轮廓，效果类似素描
- 轮廓清晰的图（黑白图、Logo、卡通图）效果最佳
- 可调 Canny 阈值控制边缘细节量

### ✏ 文字模式
- 支持多行文字（输入空行/直接回车结束）
- 自动适配画布大小调整字号
- 优先使用微软雅黑/黑体，支持中英文

---

## 参数参考

| 参数 | 说明 | 默认 |
|------|------|------|
| 画布比例 | 绘制区域占屏幕比例 | 80% |
| Canny 低阈值 | 越低边缘越多 | 50 |
| Canny 高阈值 | 越低保留更多边缘 | 150 |
| 绘制速度 | 慢/中/快 | 中速 |
| 倒计时 | 切换窗口的准备时间 | 5秒 |
