# 🎨 Auto Painter — 自动绘画程序

在杀戮尖塔2地图界面，按住右键可以自由绘画。  
本工具可以自动模拟这个过程，将**文字或图片**绘制到画面上。

---

## 安装依赖

```bash
pip install -r requirements.txt
```

## 启动

### 图形界面版（推荐）

```bash
python gui_app.py
```

### 命令行版

```bash
python main.py
```

### 打包为 .exe

```bash
python build_exe.py
```

打包完成后一个独立的 `dist/AutoPainter.exe` 即可直接运行，无需 Python 环境。

---

## 使用流程

### 图形界面版
1. 运行 `python gui_app.py` 启动程序
2. 左侧选择「图片模式」或「文字模式」选项卡
3. 图片模式：点击拖入区域选择图片，或直接拖入文件；文字模式：在文本框输入内容
4. 点击「预览效果」查看边缘检测结果
5. 右侧调整画布比例、速度、倒计时等设置
6. 点击「开始绘制」，在倒计时内切换到游戏窗口地图界面
7. 绘制过程中可随时点击「停止」中断

### 命令行版
1. 运行 `python main.py`，选择模式（图片 / 文字）
2. 按提示输入内容和参数
3. 设置倒计时秒数（默认5秒），期间切换到游戏窗口并停留在地图界面
4. 程序自动开始绘制，以屏幕中心为基准

> ⚡ **紧急停止**：把鼠标移到屏幕**左上角**立即中断

## 模式说明

### 🖼 图片模式
- 支持 jpg / png / bmp 格式
- 使用 Canny 边缘检测提取轮廓，效果类似素描
- 轮廓清晰的图（黑白图、Logo、卡通图）效果最佳
- 可调 Canny 阈值控制边缘细节量

### ✏ 文字模式
- 支持多行文字（输入空行/直接回车结束）
- 自动适配画布大小调整字号
- 优先使用微软雅黑/黑体，支持中英文

---

## 参数参考

| 参数 | 说明 | 默认 |
|------|------|------|
| 画布比例 | 绘制区域占屏幕比例 | 80% |
| Canny 低阈值 | 越低边缘越多 | 50 |
| Canny 高阈值 | 越低保留更多边缘 | 150 |
| 绘制速度 | 慢/中/快 | 中速 |
| 倒计时 | 切换窗口的准备时间 | 5秒 |

## 文件结构

```

├── gui_app.py           # 图形界面主程序（推荐）

└── requirements.txt     # 依赖列表
```
